Repo-Scribe is an experimental AI-powered tool that automatically updates documentation when code changes via GitHub pull requests.

## Why?

Documentation decays.
Code evolves.
Humans forget.

Repo-Scribe treats documentation as a living artifact.

Every PR becomes a documentation event.

## Core Features (MVP)

- Listens to GitHub pull request webhooks
- Parses code diffs
- Generates updated documentation using AI
- Posts suggested docs back to PR

## Tech Stack

- Python
- FastAPI
- GitHub Webhooks
- OpenAI-compatible LLMs (local or free-tier)
- Markdown
